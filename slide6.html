Anatomy of a Behavior Tree

STRUCTURE_ANALYSIS

DEPTH: 3 LAYERS

Root Node

The entry point. Ticked once per frame by the game engine. Passes the tick down to its child.

Composite Nodes

Selector (?): OR logic. Succeeds if any child succeeds.
Sequence (→): AND logic. Succeeds only if all children succeed.

Decorator Nodes

Wrappers that modify a single child. Examples: Inverter, Repeater, Succeeder.

Leaf Nodes

Condition: Checks state (e.g., "IsEnemyVisible?").
Action: Performs work (e.g., "MoveToTarget").
